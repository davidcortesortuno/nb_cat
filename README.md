# nb_cat

Print Jupyter notebooks on a terminal

If we execute `python nb_cat.py`:

```
--------------------------------------------------------------------------------

[M]:   # Test notebook


--------------------------------------------------------------------------------

[C]:   # Import some libraries
       import numpy as np


--------------------------------------------------------------------------------

[C]:   np.arange(7)

[Out]: array([0, 1, 2, 3, 4, 5, 6])

--------------------------------------------------------------------------------

...

```
